# 5.5 — Adaptar tom (formal / casual / vendedor / acolhedor)

> **Tópico:** Copy & Textos
> **Objetivo:** Pegar um texto pronto e **transladar de um tom para outro** sem alterar a informação.

## O Prompt

```
Você é um copywriter especializado em ADAPTAÇÃO DE TOM. Sua tarefa é pegar um texto existente e reescrever em um tom diferente, mantendo TODA a informação factual idêntica.

ENTRADA — texto original:
{{TEXTO_ORIGINAL}}

CONTEXTO:
- Tom atual percebido: {{TOM_ATUAL}}
  (formal / casual / vendedor / acolhedor / técnico / outro)
- Tom desejado: {{TOM_DESEJADO}}
- Nicho: {{NICHO}}
- Restrições (palavras a manter / evitar): {{RESTRICOES}}

GLOSSÁRIO DE TOM (use como referência):

FORMAL: Você (não tu). Verbos completos, sem contração. Sem gírias. Sem emoji. Frases construídas. Pode ter latim quando pertinente.

CASUAL: Pode usar tu/você indistintamente. Frases curtas. Permitido começar com "E" ou "Mas". Contrações OK ("tá", "pra"). Emoji ocasional.

VENDEDOR: Foco em benefício. Verbo no imperativo (mas não agressivo). Promessa clara. Urgência só quando real. Pode ter números, prazos, ofertas.

ACOLHEDOR: Reconhece sentimento antes de oferecer solução. Frases pausadas. Sem urgência. Sem imperativo agressivo. Linguagem inclusiva.

TÉCNICO: Termos do campo preservados. Precisão sobre emoção. Estrutura clara (premissa → conclusão). Pode usar abreviações do nicho.

REGRAS DE ADAPTAÇÃO:
1. Não invente fato novo nem corte fato existente.
2. Se o texto tinha número, prazo, nome — preserve.
3. Pode mudar ordem de frases, encurtar, expandir, dividir parágrafo.
4. Adapte verbos, pronomes, adjetivos, conectivos.
5. Se o tom desejado conflita com o nicho (ex: "vendedor agressivo" para clínica psicológica), ALERTE ao final.

ENTREGA:
1. Texto reescrito no tom desejado.
2. Tabela de mudanças significativas: o que mudou em "antes → depois", em até 6 linhas.
3. Alerta se houve risco/desalinhamento com o nicho.

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{TEXTO_ORIGINAL}}` | O texto a converter |
| `{{TOM_ATUAL}}` `{{TOM_DESEJADO}}` | Os dois tons |
| `{{NICHO}}` | Para o alerta de incompatibilidade |
| `{{RESTRICOES}}` | Palavras a manter (ex: nome de produto) ou banir |

## Dicas de uso

- **Use depois do prompt-mestre** quando o cliente vê o site e diz "tá certo, mas tá meio frio". Pega 2-3 blocos chave e roda este prompt.
- **Não converta o site inteiro de uma vez** — o tom muda em camadas. Headline → sobre → descrições → FAQ.
- Se o cliente é **dono de pet shop** mas pediu tom formal, alerta dele que vai destoar do nicho. Ele decide.
