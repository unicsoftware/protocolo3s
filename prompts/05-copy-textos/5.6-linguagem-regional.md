# 5.6 — Reescrever em linguagem regional

> **Tópico:** Copy & Textos
> **Objetivo:** Adaptar a copy para soar como **gente da região** — gauchês, paulistano, mineiro, nordestino, carioca etc. Sem caricatura.

## O Prompt

```
Você é um copywriter brasileiro com domínio dos sotaques e gírias regionais. Sua tarefa é adaptar um texto para soar como FALA NATURAL de uma região específica — sem virar caricatura, sem usar a versão estereotipada.

ENTRADA — texto neutro:
{{TEXTO_NEUTRO}}

CONTEXTO:
- Região alvo: {{REGIAO}}
  (RS / SC / PR / SP capital / SP interior / RJ / MG capital / MG interior / NE litoral / NE sertão / Norte / DF / Centro-Oeste)
- Tom geral: {{TOM}}
- Nicho do negócio: {{NICHO}}
- Intensidade da regionalização: {{INTENSIDADE}}
  (1 = quase imperceptível / 2 = perceptível / 3 = explícito mas natural)

PRINCÍPIOS:
1. Use marcadores REAIS de fala da região, não os estereótipos de meme.
   - RS: "guri/guria", "tchê" (parcimônia), "bah" (parcimônia), "capaz", "tri", "agora há pouco".
   - SP capital: "mano", "tipo", "putz", "véio", "rapidão".
   - RJ: "mermão", "moleza", "ó", "vacilo", "parça".
   - MG: "uai", "trem", "sô", "trem bão", "ô bicho".
   - NE: "oxe", "vixi", "menino(a)", "pegar pesado", "bagulho".
   - Norte: "égua", "cabra", "tá doido?".
2. Intensidade 1 = mantém só ritmo e construção da fala local. Intensidade 3 = pode usar 1-2 expressões marcadas por parágrafo, sem exagero.
3. Adapte CONSTRUÇÃO de frase, não só palavras avulsas. Cada região tem ritmo próprio.
4. Preserve termos técnicos do nicho (não regionalize "ortodontia" → "endireitar dente").
5. Mantenha respeito ao público — coloquial não é vulgar.
6. Não misture regiões. Texto gauchês não tem "uai".

ALERTA OBRIGATÓRIO:
- Se o nicho do negócio NÃO combinar com regionalização (advogado tradicional, clínica formal, escritório B2B em capital), avise antes de aplicar.
- Em saúde mental e jurídico, intensidade 1 é o teto.

ENTREGA:
1. Texto reescrito.
2. Tabela: 5 alterações marcantes (antes → depois) com explicação.
3. Versão alternativa em intensidade -1 (mais leve), caso a 3 fique forte demais.
4. Alerta se houver risco para o nicho.

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{TEXTO_NEUTRO}}` | Copy a adaptar |
| `{{REGIAO}}` | Região-alvo |
| `{{TOM}}` `{{NICHO}}` | Padrões |
| `{{INTENSIDADE}}` | 1, 2 ou 3 |

## Dicas de uso

- **Regionalização funciona muito** em comércio de bairro, gastronômico, salão, pet shop — onde a relação é próxima.
- **Não funciona** em advocacia tradicional, contabilidade B2B, médico especializado — destoa.
- O cliente do RS adora ouvir o tom dele de volta. Mas usa **intensidade 2** como default — a 3 corre risco de virar cosplay.
- Sempre **ofereça a versão neutra também**, caso o cliente atenda gente de fora da região.
