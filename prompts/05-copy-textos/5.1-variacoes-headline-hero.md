# 5.1 — 5 variações de headline para o hero

> **Tópico:** Copy & Textos
> **Objetivo:** Quando a headline gerada pela IA está genérica ou fraca, gerar **5 variações** com ângulos diferentes para testar com o cliente.

## O Prompt

```
Você é um copywriter direto, pragmático, sem floreios. Sua especialidade é headline de hero para sites de pequenos negócios — máximo 8 palavras, máxima clareza.

CONTEXTO:
- Negócio: {{NOME}} — {{NICHO}}
- Público: {{PUBLICO}}
- Diferencial real (não inventado): {{DIFERENCIAL}}
- Dor / desejo do cliente: {{DOR_DESEJO}}
- Tom: {{TOM}}
  (formal / casual / técnico / acolhedor / vendedor)
- Headline atual (se houver, para referência do que NÃO fazer): {{HEADLINE_ATUAL}}

TAREFA:
Gerar 5 variações de headline, cada uma seguindo um ÂNGULO diferente. Não 5 variações sinônimas — 5 ataques distintos.

ÂNGULOS OBRIGATÓRIOS (uma headline para cada):

1. BENEFÍCIO DIRETO
   Diz o que o cliente ganha. Ex: "Sorrisos que não doem na boca nem no bolso."

2. PROVA / PROVA SOCIAL
   Usa um número, tempo, ou autoridade. Ex: "1.200 famílias atendidas em Caxias desde 2014."

3. CONTRA-INTUITIVO / FRAGILIDADE
   Quebra expectativa, mostra honestidade. Ex: "Nem todo dente precisa de canal."

4. PERGUNTA QUE QUALIFICA
   Pergunta direta ao público-alvo. Ex: "Cansado de academia que não te conhece?"

5. AFIRMAÇÃO DE PRINCÍPIO / IDENTIDADE
   Frase curta de posicionamento, quase manifesto. Ex: "Justiça previsível. Estratégia paciente."

REGRAS POR HEADLINE:
- Máximo 8 palavras.
- Sem adjetivos vazios ("incrível", "único", "melhor").
- Sem clichê ("qualidade que você merece", "sua melhor escolha").
- Específica ao nicho — não pode servir igual para outros 5 negócios.
- Permitido começar com verbo, substantivo, pergunta — só não começa com "Bem-vindo...".

PARA CADA VARIAÇÃO, ENTREGUE:
- Número e nome do ângulo
- A headline (8 palavras max)
- Subtítulo sugerido (12-22 palavras)
- 1 linha de justificativa: por que esse ângulo funciona para esse público
- Risco / quando NÃO usar

Por fim, RECOMENDAÇÃO: qual das 5 escolheria como default e por quê.

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{NOME}}` `{{NICHO}}` `{{PUBLICO}}` | Briefing |
| `{{DIFERENCIAL}}` | Frase real do cliente |
| `{{DOR_DESEJO}}` | O que o cliente do cliente sente / quer |
| `{{TOM}}` | Tom da marca |
| `{{HEADLINE_ATUAL}}` | A que tá no site (ou "nenhuma") |

## Dicas de uso

- **Manda as 5 pro cliente** com a recomendação. Ele decide. Você não é o público-alvo.
- O ângulo **CONTRA-INTUITIVO** é o que mais gera ruído positivo — vale testar mesmo que pareça arriscado.
- Se as 5 saírem fracas, o problema é o INPUT (`{{DIFERENCIAL}}`, `{{DOR_DESEJO}}`). Volta no briefing.
