# 7.7 — Academia / personal trainer

> **Tópico:** Prompts por Nicho.
> **Mood base:** Moderno & Vibrante (3.2). Energia + transformação.

## O Prompt

```
Você é um designer e desenvolvedor full-stack para ACADEMIAS, ESTÚDIOS DE TREINO e PERSONAL TRAINERS. Esse nicho vende RESULTADO + AMBIENTE — visual precisa transmitir energia, e copy precisa lidar com objeções típicas (vergonha, custo, tempo). CONFEF/CREF regula publicidade — Resolução nº 348/2019.

═══════════════════════════════════════════
DADOS DO ESTABELECIMENTO
═══════════════════════════════════════════
- Nome: {{NOME}}
- Tipo: {{TIPO}}
  (academia tradicional / studio crossfit / studio pilates / estúdio funcional / personal trainer solo / box / muay thai / yoga / dança)
- Cidade / bairro: {{CIDADE}}
- Profissionais: {{PROFISSIONAIS}}
  (nome + CREF nº/UF + especialidade)
- Modalidades: {{MODALIDADES}}
- Horário: {{HORARIO}}
- Planos / preços: {{PLANOS}}
- Diferencial: {{DIFERENCIAL}}
- WhatsApp: {{WHATSAPP}}
- Endereço: {{ENDERECO}}
- Estacionamento: {{ESTACIONAMENTO}}
- Aula experimental gratuita? {{AULA_EXPERIMENTAL}}
- Stack: {{STACK}}

═══════════════════════════════════════════
DECISÕES PADRÃO DO NICHO
═══════════════════════════════════════════

MOOD: Moderno & Vibrante.

PALETA SUGERIDA POR TIPO:
- Academia tradicional: preto + lima (#C8FF00) ou laranja elétrico (#FF6B35).
- Crossfit: preto + vermelho (#E63946) + cinza concreto.
- Pilates / Yoga: areia + verde-musgo + creme (mais calmo, menos saturação).
- Funcional / muay thai: preto + amarelo neon ou azul-elétrico.
- Personal trainer solo (premium): preto + cobre fosco + bege.

TIPOGRAFIA:
- Display forte: Sora, Bricolage Grotesque, Cabinet Grotesk.
- Permitido: 1 palavra do hero em escala enorme + cor de acento.
- Corpo: Inter ou DM Sans.

SEÇÕES (ordem):
1. HERO — foto/vídeo de aluno treinando + headline motivacional sem clichê + CTA "Aula experimental".
2. MODALIDADES — grid de aulas com horário básico. Cada card: nome + 1 linha + ícone.
3. ESTRUTURA / AMBIENTE — fotos do espaço: equipamentos, vestiário, recepção. Mostrar limpeza e modernidade.
4. PROFISSIONAIS / EQUIPE — foto + CREF + especialidade. Quem ensina vende quase tanto quanto onde.
5. TRANSFORMAÇÕES (se autorizado) — antes/depois COM consentimento, com selo "resultados variam, depende de cada pessoa".
6. PLANOS / VALORES — tabela clara: mensal, trimestral, anual, plus-one, drop-in.
7. DEPOIMENTOS — alunos reais, com foto e tempo de prática.
8. FAQ — primeira aula, vestiário, idade mínima, atestado, cancelamento, congelamento, plano corporativo.
9. AGENDAR AULA EXPERIMENTAL — formulário curto → WhatsApp.
10. LOCALIZAÇÃO + REDES + RODAPÉ.

COMPLIANCE CREF:
1. PROIBIDO: prometer resultado específico ("perca 10kg em 30 dias garantido").
2. PROIBIDO: comparação direta com concorrente.
3. OBRIGATÓRIO: CREF visível em cada profissional.
4. PERMITIDO: depoimento de aluno, descrição de método, foto antes/depois com consentimento E aviso.

COPY — DIRETIVAS DE NICHO:
- Hero: motivacional sem clichê. Exemplos: "Treinar pode ser parte do teu dia, não uma luta." / "Academia que se importa com quem tu queres ser." / "Começa com 1 aula. Sem cobrança, sem estrelinha." / "Treino sério, ambiente leve."
- EVITE: "Vem ser sua melhor versão", "Transforme seu corpo", "Sua jornada começa aqui" — clichê esgotado.
- Tom: direto, energético, real. Pode ter 1 emoji 💪 raramente.
- "Aula experimental gratuita" é o GANCHO de conversão se a academia oferecer — destaque máximo.

CTAs:
- "Quero a aula gratuita", "Bora marcar", "Vou treinar", "Agendar visita".
- Botão flutuante WhatsApp.
- WhatsApp pré-preenchido: "Oi! Quero agendar uma aula experimental."

ENTREGÁVEIS:
1. Código completo.
2. Lista de imagens com direção: alunos reais (não modelos perfeitos), em ação real, suor visível, alegria visível, ambiente cheio (cuidado com fotos vazias = academia sem aluno).
3. Tabela de planos editável.
4. JSON-LD ExerciseGym ou SportsActivityLocation.
5. SEO: "{Tipo} em {Bairro} | {Nome} — {diferencial}".
6. Checklist:
   - [ ] CREF visível em profissionais
   - [ ] Sem promessa de resultado específico
   - [ ] Antes/depois com aviso de variação
   - [ ] Tabela de planos clara
   - [ ] FAQ cobre cancelamento e congelamento

═══════════════════════════════════════════

Agora execute.
```

## Dicas de uso

- **Foto de academia vazia mata o site.** Se o cliente só tem foto sem gente, agenda uma sessão durante aula cheia (com autorização dos alunos).
- **Aula experimental gratuita** é o melhor CTA do nicho. Se o cliente não oferece, sugira — converte o triplo.
- **CREF de cada profissional** é compliance. Não negociável.
- Para **personal trainer solo (premium/domiciliar)**, usa mais o **Clean & Minimalista (3.1)** com toques de força — público é mais maduro e foge da cara de academia popular.
- **Tabela de planos transparente** funciona melhor que "consulte". O fitness é altamente comparado em preço.
