# 7.8 — Imobiliária / corretor de imóveis

> **Tópico:** Prompts por Nicho.
> **Mood base:** Clean & Minimalista (3.1) com toques de elegância. CRECI obrigatório.

## O Prompt

```
Você é um designer e desenvolvedor full-stack para IMOBILIÁRIAS e CORRETORES INDEPENDENTES. O ativo principal são os IMÓVEIS — fotos boas + busca eficiente. CRECI regula publicidade — Resolução COFECI nº 1.474/2017 e Lei 6.530/78.

═══════════════════════════════════════════
DADOS DO CORRETOR / IMOBILIÁRIA
═══════════════════════════════════════════
- Nome: {{NOME}}
- Tipo: {{TIPO}} (corretor solo / imobiliária pequena / imobiliária com equipe)
- Cidade / região: {{CIDADE}}
- Corretor responsável (CRECI nº/UF): {{CRECI_RESPONSAVEL}}
- Equipe (se aplicável, nome + CRECI): {{EQUIPE}}
- Especialidade: {{ESPECIALIDADE}}
  (residencial alto padrão / popular / lançamentos / aluguel / temporada / comercial / rural / litoral)
- Diferencial: {{DIFERENCIAL}}
- WhatsApp: {{WHATSAPP}}
- Endereço: {{ENDERECO}}
- Stack: {{STACK}}

═══════════════════════════════════════════
DECISÕES PADRÃO DO NICHO
═══════════════════════════════════════════

MOOD: Clean & Minimalista, com toque de elegância (mais próximo de clássico em premium).

PALETA SUGERIDA:
- Padrão: navy (#0F1B2D) + bege quente (#F0E6D6) + branco.
- Premium / alto padrão: preto carvão + dourado fosco (#C9A961) + marfim.
- Popular / volume: azul-petróleo + branco + acento laranja para CTAs.

TIPOGRAFIA:
- Padrão: Inter / Manrope (limpo).
- Premium: Playfair Display + Inter.

SEÇÕES (ordem):
1. HERO — foto ampla de imóvel desejável + headline + busca rápida (filtros: tipo, bairro, faixa de preço).
2. IMÓVEIS EM DESTAQUE — grid de 6-12 cards com foto principal, preço, bairro, m², dormitórios, vagas. Cada card linka para detalhe (multi-página) OU abre modal (single-page).
3. POR QUE COMIGO / NÓS — diferenciais reais: tempo de mercado, conhecimento da região, processo, tipo de cliente atendido.
4. SOBRE O CORRETOR / EQUIPE — foto + CRECI + experiência + especialização.
5. SERVIÇOS — compra, venda, locação, avaliação, gestão de aluguel, financiamento.
6. FAQ — comissão, ITBI, financiamento, primeira visita, prazo, documentação.
7. CONTATO — formulário, WhatsApp, mapa.
8. RODAPÉ com CRECI.

COMPLIANCE CRECI:
1. OBRIGATÓRIO: CRECI nº do corretor responsável visível no site (rodapé + sobre + cards de imóvel quando feito por corretor específico).
2. OBRIGATÓRIO: cada anúncio de imóvel deve identificar o profissional (nome + CRECI).
3. PROIBIDO: anunciar imóvel sem autorização escrita do proprietário.
4. PROIBIDO: omitir status (à venda / vendido / sob proposta).
5. PROIBIDO: omitir informações relevantes que possam induzir o consumidor.
6. PERMITIDO: descrever o bairro, vista, infraestrutura, com fidelidade.

ESTRUTURA DE CARD DE IMÓVEL (template):
- Foto principal (cover) + 4-6 fotos no detalhe.
- Tipo (apartamento / casa / cobertura / lote / sala comercial).
- Bairro + cidade.
- Preço (sempre visível — exceto se estratégia de "consulte" for específica do nicho).
- Características: m², dormitórios, suítes, banheiros, vagas, andar.
- Status: à venda / aluguel / vendido / locado.
- CRECI do corretor responsável.
- Botão CTA: "Quero visitar" / "Saber mais" → WhatsApp pré-preenchido com referência do imóvel.

COPY — DIRETIVAS DE NICHO:
- Hero: específico para a especialidade. Exemplos:
  - Premium: "Imóveis selecionados em [bairro]. Atendimento por hora marcada."
  - Popular: "Encontra a casa nova com quem conhece o bairro."
  - Litoral: "Sua casa de praia, do verão à escritura."
- Tom: profissional, confiável, factual. Evite hipérbole ("o melhor", "imperdível", "oportunidade única").
- Descrição de imóvel: factual + 1 detalhe diferenciado. Não "lar dos seus sonhos".

CTAs:
- "Agendar visita", "Quero saber mais", "Falar com {nome}".
- WhatsApp pré-preenchido (com referência do imóvel quando aplicável):
  "Oi {nome}, vi o {tipo} em {bairro} no site (ref. {numero}). Gostaria de saber mais."

ENTREGÁVEIS:
1. Código completo, com template de card de imóvel reutilizável.
2. Lista de imagens com direção: foto profissional do imóvel — ângulo amplo, luz natural, ambiente arrumado, vista pela janela.
3. Estrutura de dados (JSON ou similar) para cadastrar imóveis facilmente — placeholder de 6 imóveis.
4. JSON-LD RealEstateAgent + Product (ou Residence) para cada imóvel.
5. SEO por imóvel quando multi-página: "{Tipo} {dorms}D em {Bairro}, {Cidade} | {Nome}".
6. Checklist:
   - [ ] CRECI no rodapé
   - [ ] CRECI em cada anúncio
   - [ ] Status do imóvel atualizado
   - [ ] Sem hipérbole no copy
   - [ ] Foto profissional ou no mínimo bem feita

═══════════════════════════════════════════

Agora execute.
```

## Dicas de uso

- **Multi-página ganha de single** quando há mais de 8 imóveis ativos. Cada imóvel vira página própria com URL amigável (`/imovel/123-apto-3d-petropolis`). Bom para SEO local.
- **Foto profissional do imóvel é obrigatória.** Foto de celular tremida desvaloriza tudo.
- **CRECI sempre visível.** Compliance.
- Se o corretor é **alto padrão / discreto**, evite filtros e listas grandes — substitua por **curadoria** (poucos imóveis, sob agendamento).
- **WhatsApp com referência do imóvel** já no link facilita a vida do corretor: ele recebe a mensagem já sabendo qual imóvel é.
