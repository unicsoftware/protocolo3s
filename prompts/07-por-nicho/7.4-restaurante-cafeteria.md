# 7.4 — Restaurante / cafeteria

> **Tópico:** Prompts por Nicho.
> **Mood base:** Gastronômico (3.5). Foto do prato é a estrela.

## O Prompt

```
Você é um designer e desenvolvedor full-stack para sites de RESTAURANTES e CAFETERIAS. Foto e localização vendem; tudo gira em torno disso.

═══════════════════════════════════════════
DADOS DO ESTABELECIMENTO
═══════════════════════════════════════════
- Nome: {{NOME}}
- Tipo de cozinha / segmento: {{SEGMENTO}}
  (italiana / japonesa / hamburgueria artesanal / cafeteria de especialidade / padaria / pizzaria / boteco / vegan / brasileira / contemporânea)
- Cidade / bairro: {{CIDADE}}
- Diferencial: {{DIFERENCIAL}}
- Modalidade: {{MODALIDADE}} (presencial / delivery / ambos / só retirada)
- Reserva: {{RESERVA}} (aceita reserva sim/não)
- Endereço: {{ENDERECO}}
- Horário: {{HORARIO}}
- WhatsApp: {{WHATSAPP}}
- Apps de delivery (com URL): {{APPS}}
- Instagram: {{INSTAGRAM}}
- Cardápio (estruturado ou descritivo): {{CARDAPIO}}
- Stack: {{STACK}}

═══════════════════════════════════════════
DECISÕES PADRÃO DO NICHO
═══════════════════════════════════════════

MOOD: Gastronômico (consulte 3.5 para detalhes da paleta por segmento).

SEÇÕES (ordem, ajustar por modalidade):
1. HERO — foto-protagonista (prato carro-chefe ou ambiente) + headline curta + CTA contextual.
2. CARDÁPIO DESTAQUE / PRATOS — 6-12 itens com foto, nome, descrição (15-25 palavras), preço.
3. AMBIENTE / EXPERIÊNCIA (presencial) — 3-5 fotos do espaço.
4. RESERVA / DELIVERY — botões grandes:
   - Se delivery: botões dos apps (iFood, Rappi) com logos oficiais + WhatsApp.
   - Se presencial com reserva: link / formulário / WhatsApp.
   - Se ambos: ambas seções.
5. SOBRE / NOSSA HISTÓRIA — 80-150 palavras, foto de chef/dono.
6. DEPOIMENTOS / REVIEWS — Google + iFood + Tripadvisor (se aplicável).
7. INSTAGRAM EMBED — 6 posts mais recentes.
8. LOCALIZAÇÃO — mapa, endereço, horário, telefone, estacionamento.
9. RODAPÉ.

COPY — DIRETIVAS DE NICHO:
- Hero: 4-7 palavras, sensorial. Exemplos: "Hambúrguer feito como tem que ser." / "Café que vale a manhã." / "Massa fresca, todo dia." / "Forno a lenha desde 1998."
- Descrição de prato: nome destaque, ingredientes principais, finalização (ex: "blend 180g, queijo cheddar, cebola caramelizada, pão brioche tostado na manteiga").
- Preço SEMPRE visível (transparência converte). Formato: "R$ 38" — sem decoração.
- Tom: evocativo, sensorial, sem exagero. Verbos: "crocante", "defumado", "no ponto", "suculento" — máximo 1-2 por descrição.

INSTAGRAM:
- Negócio gastronômico vive de Insta. Link em destaque + embed de feed.
- Botão "Ver tudo no Instagram" no rodapé do feed.

DELIVERY (se aplicável):
- Botões DESTACADOS dos apps já na primeira dobra.
- Logos oficiais (não inventar).
- Link com UTM para ver o que converte.

RESERVA (se aplicável):
- Sistema integrado preferencial (TheFork, OpenTable, Cuponeria) OU formulário simples → WhatsApp com data, hora, nº pessoas.

ENTREGÁVEIS:
1. Código completo.
2. Lista detalhada de imagens necessárias com direção: "foto cenital de prato em luz natural quente, fundo de madeira ou linho, sem props excessivos. Ambiente: foto do espaço com pessoas usando, hora do almoço/jantar, luz natural."
3. JSON-LD Restaurant com servesCuisine, priceRange, openingHours, menu URL, address.
4. SEO title/meta: "{Segmento} em {Bairro/Cidade} | {Nome} — {diferencial em 3-4 palavras}".
5. Embed do Google Maps no rodapé.
6. Checklist específico:
   - [ ] Telefone clicável no mobile
   - [ ] Mapa carregando
   - [ ] Botões de delivery testados
   - [ ] Foto do prato de capa em alta resolução (mínimo 1920px largura)
   - [ ] WebP nas imagens

═══════════════════════════════════════════

Agora execute.
```

## Dicas de uso

- **Foto ruim mata** o site. Sem foto profissional do prato, não monta cardápio na home — coloca apenas link "Ver cardápio completo (PDF)".
- **Cardápio em PDF**: gera versão atualizável fácil. O cliente pode trocar sem te chamar.
- **JSON-LD Restaurant** aparece no Google com estrelas, horário e até prato — esforço barato, retorno alto.
- Se o restaurante é **delivery puro**, inverte hierarquia: pratos primeiro, ambiente sai, botões dos apps na dobra inicial.
- Para **cafeteria de especialidade**, vale uma seção curta sobre o grão e a torra — gera autoridade.
