# 7.10 — Contador / escritório contábil

> **Tópico:** Prompts por Nicho.
> **Mood base:** Clean & Minimalista (3.1) com toques clássicos. CRC obrigatório.

## O Prompt

```
Você é um designer e desenvolvedor full-stack para CONTADORES e ESCRITÓRIOS DE CONTABILIDADE. O cliente típico é PJ que busca confiança técnica, prazo e organização — não quer "agência hype". CRC regula publicidade — Resolução CFC nº 1.307/2010 e Código de Ética CFC.

═══════════════════════════════════════════
DADOS DO ESCRITÓRIO
═══════════════════════════════════════════
- Nome: {{NOME}}
- Cidade: {{CIDADE}}
- Contadores (nome + CRC nº/UF): {{CONTADORES}}
- Tempo de mercado: {{TEMPO}}
- Especialidades: {{ESPECIALIDADES}}
  (MEI / Simples Nacional / Lucro Presumido / Lucro Real / e-commerce / startups / médicos / advogados / produtor rural / autônomos / abertura de empresa / departamento pessoal)
- Diferencial: {{DIFERENCIAL}}
- Sistema utilizado: {{SISTEMA}}
  (Domínio, Alterdata, Conta Azul, Omie, etc. — vira selo de eficiência)
- WhatsApp: {{WHATSAPP}}
- E-mail: {{EMAIL}}
- Endereço: {{ENDERECO}}
- Horário: {{HORARIO}}
- Atende remoto / online? {{REMOTO}}
- Stack: {{STACK}}

═══════════════════════════════════════════
DECISÕES PADRÃO DO NICHO
═══════════════════════════════════════════

MOOD: Clean & Minimalista, com toques clássicos para escritórios mais tradicionais.

PALETA SUGERIDA:
- Padrão moderno: navy (#0F1B2D) + branco + acento verde-petróleo.
- Tradicional: bordô ou navy + marfim + dourado fosco discreto.
- Para nichos jovens (startups, e-commerce): azul-petróleo (#1E5C7A) + branco + lima discreto.

TIPOGRAFIA:
- Padrão: Inter / Manrope (sans-serif limpa, com pesos 500/600 nos títulos).
- Tradicional: Playfair Display + Inter.

SEÇÕES (ordem):
1. HERO — headline factual + CTA "Solicitar diagnóstico" / "Falar com contador".
2. SERVIÇOS — grid claro:
   - Contabilidade mensal (Simples / Presumido / Real).
   - Departamento pessoal.
   - Imposto de renda PF.
   - Abertura de empresa.
   - Mudança de regime.
   - Consultoria fiscal.
3. ESPECIALIDADES POR SEGMENTO — destaque dos nichos atendidos (ex: "Atendemos médicos, e-commerces e produtores rurais com método específico para cada um").
4. COMO TRABALHAMOS — processo em 3-4 passos: diagnóstico, onboarding, rotina mensal, suporte.
5. SOBRE / EQUIPE — foto + CRC + experiência. Pode ser foto formal do escritório.
6. DEPOIMENTOS — clientes PJ com nome da empresa e cargo do depoente (com autorização).
7. FAQ — preço, troca de contador, prazo, primeira reunião, atendimento online, certificado digital.
8. CONTATO — formulário, WhatsApp, mapa.
9. RODAPÉ com CRC.

COMPLIANCE CRC:
1. PROIBIDO: garantir resultado fiscal específico ("vamos diminuir seu imposto em X%").
2. PROIBIDO: anúncio com base em comparação direta com concorrente.
3. PROIBIDO: oferecer "primeira consulta grátis" como atração comercial agressiva (mercantilização) — pode causar processo ético.
4. OBRIGATÓRIO: CRC visível no rodapé e em cada contador identificado.
5. PERMITIDO: apresentar formação, especialização, sistemas, áreas atendidas, depoimento de cliente sobre serviço.

COPY — DIRETIVAS DE NICHO:
- Hero: factual + benefício de processo. Exemplos: "Contabilidade que cumpre prazo." / "Seu fiscal em dia. Sua cabeça liberada." / "Contador atualizado, com sistema integrado." / "Trocar de contador é mais simples do que parece."
- Tom: profissional, calmo, sem urgência artificial.
- Sempre incluir: prazo de resposta (ex: "respondemos em até 4h em horário comercial"), modalidade (presencial e/ou online), sistemas usados.
- FAQ obrigatório: como faz para trocar de contador, primeiro mês, certificado digital, custo, integração com bancos/ERPs.

CTAs:
- "Solicitar diagnóstico", "Falar com contador", "Pedir proposta", "Tirar dúvida".
- Evite: "Bora", "Garante", "Aproveite".
- WhatsApp pré-preenchido: "Boa tarde. Vim pelo site. Gostaria de uma proposta para [tipo de empresa] do regime [Simples/Presumido/etc.]."

ENTREGÁVEIS:
1. Código completo.
2. Lista de imagens com direção: foto do escritório (ambiente organizado, computadores, equipe trabalhando), retrato profissional do contador.
3. JSON-LD AccountingService + ProfessionalService.
4. SEO: "Contador em {Cidade} | {Nome} — {especialidade}".
5. Checklist:
   - [ ] CRC no rodapé
   - [ ] CRC ao lado do nome de cada contador
   - [ ] Sem promessa de resultado fiscal
   - [ ] Sem oferta promocional agressiva
   - [ ] Processo de trabalho descrito claramente
   - [ ] FAQ cobre troca de contador

═══════════════════════════════════════════

Agora execute.
```

## Dicas de uso

- **"Como trocar de contador"** é a pergunta mais frequente — quem busca contador no Google geralmente JÁ tem um e está descontente. FAQ resolve direto a objeção.
- **Sistemas usados** (Conta Azul, Omie, Domínio) viram selo de modernidade. Inclui logos com permissão.
- **Compliance CRC**: nada de "te economizamos imposto". Pode dizer "trabalhamos para que você pague exatamente o que deve".
- Para escritório que atende **nicho específico** (médicos, e-commerce, produtor rural), faz uma seção dedicada — converte 3x melhor.
- **CRC no rodapé** é obrigatório. Cuidado com isso ao gerar.
