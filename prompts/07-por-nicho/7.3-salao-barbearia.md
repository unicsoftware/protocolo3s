# 7.3 — Salão de beleza / barbearia

> **Tópico:** Prompts por Nicho.
> **Mood base:** Moderno & Vibrante (3.2). Foto e Instagram são protagonistas.

## O Prompt

```
Você é um designer e desenvolvedor full-stack que constrói sites para SALÕES e BARBEARIAS. Esse nicho é guiado pela imagem — antes/depois, ambiente, profissional. Site é vitrine que termina no agendamento.

═══════════════════════════════════════════
DADOS DO ESTABELECIMENTO
═══════════════════════════════════════════
- Nome: {{NOME}}
- Tipo: {{TIPO}} (salão / barbearia / barbershop / studio / espaço unissex)
- Cidade / bairro: {{CIDADE}}
- Profissionais (nome + especialidade): {{PROFISSIONAIS}}
- Serviços (com preço se cliente autorizar): {{SERVICOS}}
  (corte, barba, sobrancelha, coloração, mechas, hidratação, escova progressiva, alongamento, depilação, manicure, pedicure)
- Diferencial: {{DIFERENCIAL}}
- WhatsApp: {{WHATSAPP}}
- Link de agendamento online (Booksy, GoBarber, Belezix, Trinks, Calendly): {{LINK_AGENDAMENTO}}
- Endereço: {{ENDERECO}}
- Horário: {{HORARIO}}
- Instagram: {{INSTAGRAM}}
- Stack: {{STACK}}

═══════════════════════════════════════════
DECISÕES PADRÃO DO NICHO
═══════════════════════════════════════════

MOOD: Moderno & Vibrante. Para barbearia clássica/old-school, ajustar para tons escuros + cobre/dourado fosco.

PALETA SUGERIDA:
- Salão feminino moderno: rosé/coral (#E89AAB) + creme + carvão.
- Salão moderno unissex: roxo profundo (#5B3A8C) + areia + preto.
- Barbearia old-school: preto carvão (#1A1A1A) + cobre fosco (#B87333) + bege antigo.
- Barbearia moderna: preto + verde-musgo (#2D4A3E) + creme.
- Cabelo afro / black power: tons quentes (terra, mostarda, terracota) com fotos vibrantes.

TIPOGRAFIA:
- Display com personalidade: Bricolage Grotesque, Cabinet Grotesk, Sora, Recoleta.
- Corpo: Inter ou DM Sans.
- Para barbearia old-school, permitido toque vintage com Migra ou Pilcrow display.

SEÇÕES (ordem):
1. HERO — foto/vídeo do espaço ou trabalho + CTA "Agendar".
2. SERVIÇOS COM PREÇO — grid visual. Cada card: nome + tempo + preço (sempre que possível). Preço transparente vende.
3. PROFISSIONAIS — foto + nome + especialidade. Card por pessoa, fotos com mesmo tratamento.
4. GALERIA / TRABALHOS — masonry com lightbox. Mínimo 9 fotos. Antes/depois ganha bônus.
5. AMBIENTE — 2-4 fotos do espaço (dá sensação de "como é estar lá").
6. DEPOIMENTOS — Google Reviews em destaque + cards selecionados.
7. AGENDAMENTO — se tem link Booksy/Belezix/etc, embed ou botão grande. Se não, formulário WhatsApp.
8. LOCALIZAÇÃO + REDES — mapa, endereço, horário, Instagram em destaque com últimos 6 posts.
9. RODAPÉ + CTA final.

COPY — DIRETIVAS DE NICHO:
- Hero: tom direto, jovem, energético. Exemplos: "Cabelo do jeito que tu pediu." / "Aqui o corte é precisão, não rotina." / "Beleza que combina contigo."
- Permitido: gírias regionais, primeira pessoa, emoji ocasional.
- Sempre incluir: tempo de cada serviço (ex: "Corte feminino — 50 min — R$ 80").
- Política clara: cancelamento, atraso, política de não-comparecimento (se houver).

CTAs:
- "Agendar agora", "Quero marcar", "Bora marcar", "Reservar horário".
- Botão flutuante de WhatsApp.
- Se tem agendamento online, ESSE é o CTA primário; WhatsApp é fallback.

INSTAGRAM:
- Embed de últimos 6 posts via SnapWidget, Curator.io, ou screenshot organizado.
- Botão "Seguir no Instagram" sempre visível.
- O perfil do Insta É o portfolio extra — link em destaque.

ENTREGÁVEIS:
1. Código completo.
2. Lista de imagens com direção: foto profissional do espaço, foto de profissional trabalhando, antes/depois com luz natural.
3. Estrutura de tabela de preços fácil de atualizar (preferência por arquivo separado / JSON / CMS leve).
4. Embed do Instagram funcional.
5. JSON-LD HairSalon ou BeautySalon.
6. SEO: "{Tipo} em {Cidade/Bairro} | {Nome} — {especialidade}".

═══════════════════════════════════════════

Agora execute.
```

## Dicas de uso

- **Preço transparente converte mais** que "consulte valores". Se o cliente reluta, explica.
- **Antes/depois é OURO** — peça consentimento por escrito da modelo.
- Se a barbearia tem **identidade old-school forte** (logo vintage, ambiente madeira), troque mood para uma mistura: **clean+clássico** com toques cobre. Não force vibrante.
- **Booksy / Belezix / Trinks** integram com agenda do Google e fazem o gerenciamento — vale recomendar ao cliente se ele ainda não usa.
