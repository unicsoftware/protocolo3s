# 4.1 — Hero impactante (título + subtítulo + WhatsApp)

> **Tópico:** Componentes & Seções
> **Objetivo:** Construir ou refazer **só o hero** — o primeiro bloco que decide se o visitante rola ou fecha a aba.

---

## Quando usar

Site existente com hero fraco (headline genérica, foto ruim, CTA escondido). Ou site novo onde o hero precisa ser construído antes do resto.

## Como usar

1. Cole o tom + nicho + diferencial.
2. Indique o mood do site (clean / vibrante / clássico / acolhedor / gastronômico).
3. Saída: HTML/JSX do hero pronto + 3 variações de headline + 2 variações de subtítulo + descrição da imagem ideal.

## O Prompt

```
Você é um especialista em construção de heroes — o primeiro bloco visível de um site, que tem 3 segundos para reter o visitante. Seu trabalho é entregar UM hero, não 5 alternativas.

CONTEXTO:
- Negócio: {{NOME}} — {{NICHO}} em {{CIDADE}}
- Diferencial: {{DIFERENCIAL}}
- Público-alvo: {{PUBLICO}}
- Mood do site: {{MOOD}}
  (clean / vibrante / clássico / acolhedor / gastronômico)
- Ação principal: {{ACAO_PRINCIPAL}}
- WhatsApp pré-preenchido: {{LINK_WHATSAPP}}
- Stack: {{STACK}}
  (HTML+Tailwind / React+Tailwind / Next.js)

REGRAS DO HERO:
1. Headline: máximo 8 palavras. Não pode ser genérica ("Sua melhor escolha", "Qualidade e confiança"). Deve dizer O QUÊ + PARA QUEM ou O QUÊ + DIFERENCIAL.
2. Subtítulo: 1 frase, 12-22 palavras. Reforça com prova ou contexto, não repete.
3. CTA primário: botão grande, microcopy direta no formato "{verbo} {coisa}" — não "Saiba mais".
4. CTA secundário (opcional): link/ghost button para quem não está pronto.
5. Imagem: se for foto, deve ser do produto/serviço REAL ou contexto humano relevante. Sem mockup genérico.
6. Sinal de prova rápida: 1 elemento (ex: "+500 clientes", "9 anos no bairro", "4.9 ★ no Google", certificação relevante) — discreto, abaixo do CTA.

ANTI-PADRÕES (não faça):
- Headline com 3 adjetivos.
- "Bem-vindo ao site da..."
- Vídeo background pesado.
- Carrossel de slogans.
- Headline em CAIXA ALTA (exceto 1 palavra para ênfase em mood vibrante).

SAÍDA:
1. CÓDIGO do hero pronto, na stack indicada.
2. 3 variações alternativas de headline (numeradas), com 1 linha de justificativa cada.
3. 2 variações de subtítulo.
4. Direção de imagem em 2 frases (estilo, enquadramento, luz).
5. Microcopy do CTA primário (final) + 2 alternativas para teste.

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{NOME}}` `{{NICHO}}` `{{CIDADE}}` `{{DIFERENCIAL}}` `{{PUBLICO}}` | Briefing |
| `{{MOOD}}` | Um dos 5 moods |
| `{{ACAO_PRINCIPAL}}` | A do 2.2 |
| `{{LINK_WHATSAPP}}` | URL pronta do 2.5 |
| `{{STACK}}` | Tecnologia |

## Dicas de uso

- O **anti-padrão "Bem-vindo ao site da..."** é o mais frequente. Audite todo hero gerado pela IA contra essa lista.
- 3 segundos: se em 3 segundos eu não sei **o que é** + **para quem** + **o que clicar**, o hero falhou.
- Use as **variações alternativas** para teste com o cliente. Não decida sozinho qual é a melhor — manda as 3 pra ele.
