# 1.3 — Inferir nicho, tom e público a partir do Instagram

> **Tópico:** Descoberta & Briefing
> **Objetivo:** Pegar prints / bio / últimos posts do Instagram do cliente e extrair **nicho real, tom de voz, público que ele já atrai e identidade visual atual** — sem precisar perguntar tudo de novo.

---

## Quando usar

Cliente é meio devagar pra responder ou é do tipo "faz aí o que tu acha melhor". Antes de insistir no briefing, você abre o Instagram dele, tira 5–10 prints (bio, 6 últimos posts, 3 stories destacados, comentários), cola aqui, e o Claude te devolve uma leitura completa.

Também é ótimo para **validar** o que o cliente disse — às vezes ele se descreve como "moderno" mas o feed grita "anos 2010".

## Como usar

1. Abra o Instagram do cliente (Web facilita).
2. Cole no Claude:
   - **Bio completa** (texto)
   - **Legendas dos 6 últimos posts** (texto)
   - **Texto de 3 highlights** se tiver (ou descrição do que aparece)
   - Opcional: anexe prints. O Claude lê imagens.
3. Preencha as variáveis.
4. Saída: relatório em 7 blocos, com nível de confiança em cada inferência.

## O Prompt

```
Você é um estrategista de marca que faz leitura rápida de perfis de Instagram de pequenos negócios para extrair sinais úteis para construção de site.

DADOS DO PERFIL:

@{{HANDLE_INSTAGRAM}}

BIO:
{{BIO}}

ÚLTIMOS POSTS (legendas e descrição visual):
{{POSTS}}

HIGHLIGHTS (se houver):
{{HIGHLIGHTS}}

OUTRAS OBSERVAÇÕES (engajamento, comentários típicos, frequência de postagem):
{{OBSERVACOES}}

SUA TAREFA:
Produzir um relatório de leitura do perfil que serve como input para criação de site. Cada inferência deve vir com nível de confiança: ALTA / MÉDIA / BAIXA.

REGRAS:
1. Diferencie SINAL (o que está claramente no perfil) de SUPOSIÇÃO (o que você está deduzindo). Suposição precisa de [confiança: BAIXA] e justificativa curta.
2. Não invente serviços que não aparecem.
3. Se o perfil for inconsistente (ex: bio formal + posts em meme), aponte essa inconsistência — é informação valiosa para definir o tom do site.
4. Foque no que serve para o site, não em consultoria de Instagram.

FORMATO DE SAÍDA:

## 1. Nicho real
[o que ele de fato vende, em uma frase, com confiança]

## 2. Subnicho / posicionamento
[ex: "barbearia popular" vs "barbearia premium" — com confiança]

## 3. Tom de voz observado
[3-5 adjetivos com exemplos de legendas que comprovam — ex: "informal e bem-humorado, vide post X"]

## 4. Público que ele já atrai
[idade aproximada, gênero predominante, classe social aparente, baseado em quem comenta e em quem aparece nas fotos]

## 5. Identidade visual atual
- Paleta dominante:
- Estilo de foto (estúdio / amador / mockup / pessoa real):
- Uso de logo:
- Consistência (alta/média/baixa):

## 6. Diferenciais que aparecem no perfil
[o que ele já comunica como sendo o "porquê escolher ele"]

## 7. Inconsistências e oportunidades
[o que está bagunçado e o que o site pode resolver / amplificar]

## 8. Recomendação de tom para o site
[em uma frase: o site deve seguir o tom do Insta? alinhar? elevar? — com justificativa]

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{HANDLE_INSTAGRAM}}` | @ do perfil |
| `{{BIO}}` | Texto bruto da bio (incluindo emojis e links) |
| `{{POSTS}}` | Cole 6 legendas + descrição visual breve de cada post (1 linha cada) |
| `{{HIGHLIGHTS}}` | Nomes e tema dos destaques |
| `{{OBSERVACOES}}` | Engajamento médio, tipo de comentário, frequência |

## Dicas de uso

- **Anexe prints reais** — o Claude lê. Vale mais que descrever a foto.
- Se o cliente tem **TikTok mais ativo que o Instagram**, faz o mesmo prompt trocando a fonte. Os blocos continuam funcionando.
- Use o **Bloco 8 (Recomendação de tom)** para decidir entre os 5 prompts-mestre da etapa 3.
- Se a confiança ficar BAIXA em vários blocos, é sinal que **o perfil é fraco** — peça para o cliente mandar mais material direto.
