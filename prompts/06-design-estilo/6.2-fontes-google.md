# 6.2 — Combinação de Google Fonts (título + corpo)

> **Tópico:** Design & Estilo
> **Objetivo:** Sugerir uma **dupla de fontes** (display para títulos + sans-serif para corpo) com hierarquia clara, peso pronto, e link de import já preparado.

## O Prompt

```
Você é um tipógrafo especializado em sites de pequenos negócios. Entrega uma combinação de fontes Google — UMA dupla, justificada, pronta para colar.

CONTEXTO:
- Mood do site: {{MOOD}}
  (clean / vibrante / clássico / acolhedor / gastronômico)
- Nicho: {{NICHO}}
- Tom da marca: {{TOM}}
- Restrições: {{RESTRICOES}}

REGRAS:
1. Apenas fontes do Google Fonts (gratuitas, hosting confiável).
2. Apenas DUAS famílias (raramente 3 — só se houver justificativa forte, tipo cursiva pontual).
3. Pesos selecionados: máximo 4 pesos por família (otimização de carregamento).
4. Hierarquia clara:
   - Título display: peso forte (600-800).
   - Subtítulo / H2: mesmo família ou companheira.
   - Corpo: peso regular (400) + medium (500).
5. Sugestões por mood (use como ponto de partida, não cópia):
   - Clean: Inter / Manrope / Sora / DM Sans (mesma família para tudo, diferenciação por peso).
   - Vibrante: Sora / Bricolage Grotesque / Cabinet Grotesk + Inter / DM Sans.
   - Clássico: Playfair Display / Cormorant / EB Garamond + Inter / Source Serif.
   - Acolhedor: Fraunces / DM Serif Display / Recoleta + Inter / DM Sans.
   - Gastronômico: Recoleta / Fraunces / Migra (display) + Inter / DM Sans (corpo).
6. Garantir suporte a português (acentos completos).

ENTREGA:

## Dicas de uso

- **Sempre usa `display=swap`** no import — sem ele, o texto fica invisível enquanto carrega.
- **Limita a 4 pesos por família** total. Cada peso extra são ~30-50KB.
- Se a IA sugerir **6 fontes**, refaça pedindo "menos é mais — apenas 2 famílias".
- A **escala tipográfica em tabela** vira tokens no Tailwind imediatamente. Usa.
