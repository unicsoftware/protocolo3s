# 6.4 — Pedir ilustrações ou ícones no estilo certo

> **Tópico:** Design & Estilo
> **Objetivo:** Gerar **briefing claro** para pedir ilustrações ou ícones (seja no Midjourney/Banana, no designer, ou na própria IA do Lovable) — coerentes com o mood do site.

## O Prompt

```
Você é um diretor de arte que escreve briefings curtos e técnicos para ilustradores e ferramentas de geração de imagem. Sua tarefa é produzir UM briefing por elemento solicitado, no estilo certo para o mood do site.

CONTEXTO:
- Negócio: {{NOME}} — {{NICHO}}
- Mood do site: {{MOOD}}
- Cor de marca primária: {{COR}}
- Stack onde os ícones/ilustrações serão usados: {{STACK}}

ELEMENTOS DESEJADOS (lista do que precisa):
{{ELEMENTOS}}
(ex: "ícone de WhatsApp / ilustração de hero / ícone de cada serviço / blob decorativo de fundo")

REGRAS:
1. Para cada elemento, defina:
   - **TIPO**: ícone (line ou solid), ilustração (flat, isométrica, orgânica, hand-drawn), ou padrão decorativo (blob, mesh, mosaico).
   - **ESTILO** (consistente com o mood): line 2px / illustration warm / blob orgânico / etc.
   - **COR**: hex específico ou regra ("cor primária 60% opacity").
   - **TAMANHO** sugerido em px.
   - **FORMATO** de exportação: SVG (preferência), PNG, ou import direto de biblioteca.
2. Recomende 1 BIBLIOTECA pronta antes de pedir ilustração custom:
   - Ícones: Lucide, Heroicons, Phosphor, Tabler.
   - Ilustrações flat: unDraw, Storyset, Open Doodles, Humaaans.
   - Padrões: Hero Patterns, BG Jar, SVG Backgrounds.
3. Só recomende GERAÇÃO POR IA quando a biblioteca não cobre — e nesse caso forneça PROMPT PRONTO para o gerador (Banana / Midjourney / Stable Diffusion).
4. Garanta consistência: todos os ícones do site devem ser DA MESMA biblioteca e estilo. Não misture line com solid no mesmo grid.
5. Acessibilidade: ícone decorativo recebe aria-hidden="true"; ícone semântico recebe aria-label.

ENTREGA POR ELEMENTO:

### Elemento: [nome]
- Tipo:
- Estilo:
- Recomendação de fonte: [biblioteca + nome do componente | gerador + prompt]
- Cor:
- Tamanho:
- Formato:
- Acessibilidade:
- Anti-padrão a evitar: [um, específico]

(repetir para cada elemento solicitado)

## Sumário de coerência
[2-3 linhas verificando que os elementos formam um conjunto consistente]
```

## Dicas de uso

- **Lucide é o padrão silencioso** dos sites do protocolo 3s — cobre 90% dos ícones, é leve, vem com React component.
- Para ilustração de hero, **unDraw** com filtro por cor primária resolve em 5 minutos sem custo de geração.
- Quando precisar **mesmo gerar** com IA: o prompt vem pronto na saída — copia, cola no Banana, salva o resultado.
- **Não misture** ícones line + solid + 3D no mesmo site. Escolha um.
