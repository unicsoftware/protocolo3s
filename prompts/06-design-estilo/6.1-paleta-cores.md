# 6.1 — Paleta de 4 cores baseada no nicho + Instagram

> **Tópico:** Design & Estilo
> **Objetivo:** Sugerir uma paleta enxuta (4 cores) que **case com o nicho**, dialogue com o Instagram do cliente, e seja aplicável imediatamente (com hex + uso).

## O Prompt

```
Você é um designer de identidade visual para pequenos negócios. Sua tarefa é entregar UMA paleta de 4 cores — específica, justificada, com hex e papel definido para cada cor.

CONTEXTO:
- Negócio: {{NOME}} — {{NICHO}}
- Mood do site: {{MOOD}}
  (clean / vibrante / clássico / acolhedor / gastronômico)
- Cores que o cliente JÁ usa (logo, Instagram, ambiente físico): {{CORES_EXISTENTES}}
- Restrições / preferências do cliente: {{RESTRICOES}}
  (ex: "não quero vermelho", "tem que combinar com a cor da fachada")
- Concorrentes da praça (cores que predominam no nicho local): {{CORES_CONCORRENTES}}

REGRAS:
1. 4 cores no total, com PAPÉIS distintos:
   - PRIMÁRIA (cor de marca, usada em CTAs e elementos críticos)
   - NEUTRA CLARA (background principal — branco puro, off-white quente, marfim, areia)
   - NEUTRA ESCURA (texto principal — quase preto, marrom escuro, navy)
   - ACENTO (uso pontual — links, ícones de destaque, hover, pequenos elementos)
2. Forneça hex EXATO para cada cor.
3. Justifique cada escolha em 1-2 linhas.
4. Indique que combinações são SAFE para texto (contraste AA mínimo): liste 3-5 combinações texto sobre fundo válidas.
5. Aponte 1-2 combinações a EVITAR (baixo contraste, conflito).
6. Se o cliente já usa cores, derive a paleta delas (não substitua a cor de logo). Apenas refine, harmonize, ou expanda com neutros adequados.
7. Adapte ao mood:
   - Clean: 1 cor de acento, 90% neutros.
   - Vibrante: cor primária saturada, neutros sólidos (branco/preto pleno).
   - Clássico: navy/bordô + marfim + dourado fosco.
   - Acolhedor: tons quentes pastel, baixa saturação.
   - Gastronômico: depende do segmento (ver 3.5).

ENTREGA:

## Paleta sugerida

| Papel | Nome | Hex | RGB | Quando usar |
|---|---|---|---|---|

## Justificativa
[5-8 linhas: por que essa paleta para esse nicho + esse cliente]

## Combinações de contraste
- Texto + fundo válidos (AA): [lista]
- Evitar: [lista]

## Variações de tom
- Versão hover/active de cada cor primária e acento (10% mais escuro/claro).
- Versão para fundo de seção alternada (very light tint da primária).

## Tokens prontos (CSS variables)
```css
:root {
  --color-primary: #...;
  --color-primary-hover: #...;
  --color-bg: #...;
  --color-bg-alt: #...;
  --color-text: #...;
  --color-text-muted: #...;
  --color-accent: #...;
}
```

## Acessibilidade
- Cor primária + branco: contraste AA? [resultado]
- Cor primária + neutra escura: [resultado]
- Texto principal + bg: [resultado]

Agora execute.
```

## Variáveis

| Variável | O que preencher |
|---|---|
| `{{NOME}}` `{{NICHO}}` `{{MOOD}}` | Padrões |
| `{{CORES_EXISTENTES}}` | Hex se você sabe; ou "logo é vermelho coral, Insta tem muito amarelo" |
| `{{RESTRICOES}}` | Limitações |
| `{{CORES_CONCORRENTES}}` | Output do prompt 1.4 |

## Dicas de uso

- **A cor é onde o cliente mais opina.** Mande a paleta com mockup pequeno (hero + um botão) — ele aprova mais rápido vendo aplicado.
- Se o cliente tem **logo colorida em conflito** (ex: 5 cores), pega só 1-2 dela e deriva o resto.
- Os **CSS variables prontos** colam direto no Tailwind config ou no `<style>` do prompt-mestre.
- **Cor primária satura demais?** Reduz HSL em 10-15% de saturação. Especialmente em vibrante.
